## 1. 目标与边界

S2阶段目标：在 2026-05-20 前完成 M2 里程碑，交付以下内容。

## 6. 风险与应急机制

主要风险：
- 扫描件OCR质量波动大
- PDF图片分离失败率高
- 批处理性能不稳定

应急策略：
- 先保证原生PDF通过率，再扩展扫描件
- 采用失败重试与错误归档机制
- 每周发布误识别Top问题并集中修复
