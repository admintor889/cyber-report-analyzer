## 6. 风险与应急机制

主要风险：
- 规则与模型结论冲突
- 待复核量激增影响交付节奏
- 结果追溯链路不稳定

应急策略：
- 坚持规则优先，模型仅复核与解释
- 设定待复核阈值并分级处理
- 每周固定追溯链路专项回归
