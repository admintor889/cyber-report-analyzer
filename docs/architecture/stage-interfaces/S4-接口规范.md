# S4 接口规范（Word 输出一致性版）

## 1. 阶段目标

S4 负责把 S3 的结果接入 Web 展示和 Word 导出。页面、筛选、证据查看、规则维护和导出必须使用同一条款级 view model，Word 是最终审查材料。

## 2. 数据结构

### 2.1 QueryFilter

```json
{
  "verdict": "REVIEW",
  "field": "crypto.rsa.key_length",
  "page": 3,
  "keyword": "RSA",
  "rule_version": "2026.06"
}
```

### 2.2 ResultViewItem

```json
{
  "rule_id": "R-RSA-001",
  "rule_version": "2026.06",
  "standard_id": "S1-1.2",
  "check_item": "不合适的加密算法",
  "field": "crypto.rsa.key_length",
  "value": "2048",
  "verdict": "REVIEW",
  "reason": "RSA 2048 requires review",
  "page": 3,
  "paragraph_id": "1.2-P03",
  "section_id": "1.2",
  "snippet": "RSA2048",
  "image_ref": "img-0001",
  "source_type": "image_ocr"
}
```

### 2.3 RuleAdminItem

```json
{
  "rule_id": "R-RSA-001",
  "rule_version": "2026.06",
  "standard_id": "S1-1.2",

```

### 2.4 ExportPayload

```json
{
  "task_id": "task-0001",
  "format": "word",
  "include_evidence": true,
  "include_review": true,
  "include_rule_admin": false
}
```

## 3. 模块接口

### 3.1 web

#### list_results(task_id: str, filters: QueryFilter) -> list[ResultViewItem]
#### get_evidence(task_id: str, rule_id: str) -> dict[str, object]
#### list_rules(task_id: str | None = None) -> list[RuleAdminItem]
#### upsert_rule(payload: dict[str, object]) -> RuleAdminItem
#### toggle_rule(rule_id: str, active: bool) -> RuleAdminItem
#### export_result(payload: ExportPayload) -> dict[str, object]

### 3.2 evidence

#### build_view_model(result: dict[str, object], traces: list[dict[str, object]]) -> ResultViewItem
#### validate_trace_consistency(results: list[dict[str, object]], traces: list[dict[str, object]]) -> dict[str, object]

### 3.3 reporting

#### export_summary(results: list[dict[str, str]]) -> dict[str, int]
#### export_detail(results: list[ResultViewItem], summary: dict[str, int]) -> dict[str, object]

## 4. Word 输出要求

- Word 导出必须和页面展示字段一致。
- 一条 finding 可以展开成多行，但主键和 verdict 不得变化。
- Word 至少包含段落级编号、章节序号、关键词、标准项、判定结果、说明解释、页码和溯源点。
- 保存后的规则必须立即影响页面和导出。

## 5. 退出标准

S4 结束时，页面和 Word 输出必须完全同源，不能分别计算、分别解释。
