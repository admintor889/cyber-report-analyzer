# S1 接口规范（2026-03-31 至 2026-04-20）

## 1. 阶段目标

S1 目标是建立可联调的最小接口闭环，覆盖任务提交、基础解析、语义归一化和结果回填。

## 2. 统一数据结构

### 2.1 Task（任务对象）

```json
{
  "task_id": "task-0001",
  "file_name": "report.pdf",
  "status": "queued",
  "result": null
}
```

字段类型：
- task_id: string
- file_name: string
- status: string（queued | completed）
- result: object | null

### 2.2 SemanticReviewResult（语义归一化结果）

```json
{
  "normalized": "RSA-2048",
  "explanation": "RSA key length meets minimum baseline but should be reviewed by policy.",
  "confidence": "0.90"
}
```

字段类型：
- normalized: string
- explanation: string
- confidence: string（0~1）

## 3. 模块接口

## 3.1 web

### health() -> dict[str, str]

输入：无

输出示例：
```json
{"status": "ok"}
```

### submit_report(file_name: str) -> Task

输入：
- file_name: string，非空

输出：Task

异常：
- ValueError（file_name 为空或非字符串）

### get_task(task_id: str) -> Task

输入：
- task_id: string

输出：Task

异常：
- KeyError（task_id 不存在）

### set_task_result(task_id: str, result: dict[str, object]) -> Task

输入：
- task_id: string
- result: object

输出：更新后的 Task，status 自动更新为 completed

## 3.2 parser

### extract_text_and_images(pdf_path: pathlib.Path) -> dict[str, list[str]]

输入：
- pdf_path: pathlib.Path

输出：
```json
{
  "text_blocks": [],
  "image_paths": []
}
```

异常：
- TypeError（pdf_path 不是 Path）

## 3.3 ocr

### run_ocr(image_path: pathlib.Path) -> dict[str, str]

输入：
- image_path: pathlib.Path

输出：
```json
{
  "raw_text": "",
  "normalized_text": ""
}
```

异常：
- TypeError（image_path 不是 Path）

## 3.4 model_review

### semantic_review(item: dict[str, str]) -> SemanticReviewResult

输入优先级：
- value > raw_text > text

输出：SemanticReviewResult

异常：
- TypeError（item 不是 dict）

规则覆盖：
- RSA（RSA2048、2048-bit RSA）
- TLS（TLS1.0/1.1/1.2/1.3）
- 弱算法关键词（MD5、SHA-1、DES、3DES、RC4、ECB）

## 3.5 storage

### InMemoryRepository.add(item: dict[str, str]) -> None

输入：
- item: dict[str, str]

输出：无

### InMemoryRepository.all() -> list[dict[str, str]]

输入：无

输出：列表

## 4. S1 联调最小结果结构

```json
{
  "task_id": "task-0001",
  "fields": {
    "crypto.rsa": "2048"
  },
  "summary": {
    "PASS": 0,
    "FAIL": 0,
    "REVIEW": 1
  }
}
```

## 5. 兼容性

- S2 起开始引入版本化字段建议（schema_version）。
